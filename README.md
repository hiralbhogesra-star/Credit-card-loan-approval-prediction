#Credit Card Approval Prediction System
 Project Overview

This project is a Machine Learning-based system that predicts whether a credit card application will be Approved or Rejected.
It uses classification algorithms like Logistic Regression and Random Forest and compares their performance using various evaluation techniques.

 Fatures:
Data preprocessing (handling missing values & duplicates)
Model training using:
Logistic Regression
Random Forest Classifier
Model evaluation using:
Accuracy Score
Confusion Matrix
Classification Report
Cross Validation (CV = 5)
K-Fold Cross Validation
Hyperparameter Tuning using GridSearchCV
Prediction visualization (Actual vs Predicted)
Real-time prediction for new applicant data

Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
